P2P加密聊天系统 - 前后端不分离

系统已成功启动！

访问方式：
1. 打开浏览器
2. 访问地址：http://localhost:5000
3. 系统将显示Aurora Chat登录界面

功能说明：
1. 用户注册与登录
   - 输入用户名即可登录
   - 系统支持多用户同时在线

2. 公共聊天室
   - 所有登录用户可以在公共聊天室发送消息
   - 实时显示在线用户列表

3. P2P加密连接
   - 可以从在线用户列表中选择用户建立P2P连接
   - 支持P2P连接请求的接受/拒绝
   - 建立P2P连接后可以进行私密聊天

4. 加密功能
   - 系统使用端到端加密（ECDH密钥交换 + AES-GCM加密）
   - 所有P2P消息都经过加密传输

系统架构：
- 后端：Flask + Socket.IO
- 前端：HTML + JavaScript + Tailwind CSS
- 数据库：SQLite（自动创建）
- 加密：cryptography库

核心模块：
1. crypto.py - 加密模块（密钥生成、加密、解密）
2. p2p_connection.py - P2P连接管理
3. database.py - 数据库管理
4. app.py - 主Flask应用

测试方法：
1. 打开第一个浏览器标签页，访问 http://localhost:5000
2. 使用用户名"用户1"登录
3. 打开第二个浏览器标签页，访问 http://localhost:5000
4. 使用用户名"用户2"登录
5. 在用户1的界面中，从在线用户列表选择"用户2"，点击"建立P2P连接"
6. 用户2会收到连接请求，点击"接受"
7. 双方建立P2P连接后，可以进行私密聊天

注意事项：
1. 系统使用WebSocket进行实时通信
2. P2P连接使用UDP端口（自动分配）
3. 所有消息都经过加密处理
4. 用户数据存储在本地SQLite数据库

系统状态：
- WebSocket服务器：运行中 (ws://localhost:5000)
- HTTP服务器：运行中 (http://localhost:5000)
- P2P服务：运行中 (UDP端口: 自动分配)
- 数据库：已初始化

如需停止服务器，请在终端按 Ctrl+C
